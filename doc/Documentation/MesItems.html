Module Mes Items

Ce module permet de réunir plusieurs items existants dans une liste, puis de copier cette liste d'items dans une autre soumission existante, nommée la soumission cible. Les items et les soumissions doivent exister au préalable.

  1. Pour accéder au module Mes Items, cliquer tout d'abord sur le lien Mes Items 
  2. Le formulaire de Connexion apparaît : . Entrer le nom d'usager (et mot de passe s'il y a lieu).
  3. La page qui s'affiche est alors la liste de Mes Items (les items choisis que l'on veut copier). Cette liste peut être vide si c'est la première fois que l'usager utilise le module ou s'il a vidé sa liste : 
  4. Appuyer sur le bouton Rechercher Items pour ajouter de nouveaux items.
  5. Voici la page de recherche d'items.  Les items peuvent être recherchés par Item si on connaît déjà le nom, la description ou le numéro de l'item. Les items peuvent aussi être recherchés par Soumission, si on connaît plutôt le numéro, le nom de Projet ou le Client de la Soumission dont on veut copier les items. Cliquer sur le bouton radio qui correspond au genre de recherche souhaité (par Item ou par Soumission).
  6. Dans la case Entrer mot à chercher, entrer le mot clé qui permet d'identifier l'item : il peut s'agir d'une partie ou de tout le numéro, le nom d'item, la description de l'item ou la famille. Ces champs sont disponibles dans la liste déroulante à droite de la case du mot à chercher et ils correspondent aux champs décrivant un Item dans Access.
  7. Si on a choisi la recherche Par Soumission, on peut décider de chercher le mot clé dans les champs suivants : Numéro de Soumission, Projet, Client ou les trois à la fois.
  8. En tapant le mot clé, la recherche débute et les résultats sont affichés au-dessous dans une liste d'items: 
  9. Si on a cherché par Soumission, on obtient une liste de soumissions qui incluent le mot clé, et puis juste en-dessous, la liste des items de la soumission sélectionnée. En sélectionnant une soumission différente, la liste d'items va se mettre à jour automatiquement. 

    10. NOTE IMPORTANTE CONCERNANT LES RÉSULTATS RETOURNÉS : afin de rendre la recherche plus rapide, on a limité à 20 le nombre de résultats retournés : il est donc essentiel de faire une recherche la plus précise possible pour trouver le bon item ou la bonne soumission. Par exemple, si on cherche le mot affiche, il est clair que dans la base de données il y a beaucoup plus que 20 items qui contiennent ce mot ! Mais les résultats affichés ne vont retourner que les 20 plus récents items ayant ce mot. Si on veut une affiche qui date d'il y a plusieurs mois, il vaut mieux alors faire une recherche par soumission en retrouvant la soumission qui décrivait cet item affiche précis. À noter aussi que la recherche de plusieurs mots ne fonctionne pas, à moins que les mots soient réellement l'un à la suite de l'autre. Par exemple, pour trouver un item appelé Affiche de la semaine verte, les mots clés affiche verte ne retourneront pas cet item (ces deux mots ne sont pas consécutifs dans l'item), tandis que les mots clés semaine verte (mots consécutifs) vont bien retourner cet item.

  10. Il suffit ensuite de cocher les items que l'on souhaite puis de cliquer sur le bouton Ajouter à Mes Items 
  11. Ceci nous ramène à la page contenant la liste Mes Items. On peut y voir les items cochés précedemment. On peut ajouter d'autres items (bouton Rechercher Items). On peut également effacer des items en cliquant sur l'icône poubelle à gauche de chaque item. Pour effacer toute la liste d'un coup, cliquer sur le bouton Effacer tout à droite de la page. 
  12. Une fois que les items sont choisis, on peut rechercher une soumission (dite la cible) dans laquelle copier cette liste d'items:
  13. Entrer le mot clé dans l'espace prévu et sélectionner le champ de recherche : Numéro de soumission (par défaut), Projet, Client ou Tout pour chercher dans les trois champs à la fois.
  14. En tapant le mot clé, la recherche débute et les résultats sont affichés en-dessous sous la forme d'une liste déroulante de soumissions.
  15. Sélectionner la soumission voulue comme cible, puis cliquer sur le bouton Copier Mes Items.
  16. La page de confirmation apparaît, qui résume l'opération qui va se produire. Si tout est conforme, cliquer sur Copier; sinon, pour revenir en arrière, cliquer sur Annuler. 
  17. On revient alors à la page de la liste Mes Items; un message confirmant l'opération est affiché (succès ou erreur selon ce qui s'est passé).
